== WHAT’S INCLUDED IN OUR FREE ROLE BASED PRICING FOR WOOCOMMERCE PLUGIN ==

**1) Add-To-Cart Button & Tax Label**
Role-Based Pricing for WooCommerce plugin allows the admin to **remove or replace** the **Add-to-cart** button for guest users or other desired user roles.
Similarly, using WooCommerce role based pricing you can hide or display the tax label for different user roles.  

**2) Text Replacement for Add-To-Cart Button**
This WooCommerce user role pricing plugin, allows the admin to show text in place of the add-to-cart button. If the admin disables the add-to-cart button, this text replacement will appear instead of that button.

**3) Role based price for WooCommerce with Priority-Based Pricing Rules**
You can create priority-based pricing rules for different user role based. The admin can give priority to the created rules for selected user roles. The WooCommerce role based pricing plugin will then allow the priority rule to be applied first, then the second, and so on. However, the price rule application will dominate over priority rules.  

**4) Multiple Rule On Single User Role**
The admin can apply multiple rules on a particular user role using our role-based pricing plugin. For example, you can create three price rules for a subscriber and apply all these three price rules on the subscriber, user role. 
   
**5) Price Rule Application Type**
When multiple rules are applied to a single user role, the admin can decide what type of price rule it should act upon. The Role-based pricing for WooCommerce plugin allows the admin to select which price type he wants to apply the price rule. 

It has the option to apply price rules using: 

* Minimum Discount: The price rule will act on the rule that has the minimum discount for that user role. 
* Maximum Discount: The price rule will act on the rule that has the maximum discount value for that user role.      
* All Discounts Combined: The price rule will combine all discounts offered for a particular user role.
* Minimum Priority: The price rule will act on the price rule that the admin has set the minimum priority. 
* Maximum Priority: The price rule will act on the price that the admin has set the maximum priority.         

By default, the all Discount Combined option is checked.  

**6) Unlimited User Role Based Pricing for WooCommerce Rules**
With a user role based pricing plugin, you can create unlimited WooCommerce individual pricing rules and apply them to the regular price or sale price as desired. 

Moreover, you can apply WooCommerce user group discounts or restrict specific products, tags, or categories for WooCommerce individual pricing. It supports user role based WooCommerce category visibility. 

**7) Customer Specific Pricing for WooCommerce With Hiding Price According To User-Roles** 
WooCommerce Role Based Pricing plugin lets the store owner hide specific product prices from particular user roles. Admin can completely adjust pricing for user roles that helps in WooCommerce price management.

**8) Enable/ Disable User Role-Based Price for WooCommerce Products**  
Role-Based Pricing for WooCommerce plugin enables the admin to show or hide the WooCommerce prices by user role. Hide or show the Regular Price and Sale price to selected users. Settings for the priority of prices by user role for WooCommerce have been provided.

**9) Price Rules List**
The Price Rule List is for the admin that displays all the price rules based on user roles. You can find all the role-based pricing rules details like discount type, price, priority, date published, etc. Furthermore, the admin can create new price rules and enable or disable any price rule on your WooCommerce store.   
    
**10) Sample Price Rule To Get You Started**
When installing the MWB Role Based Pricing plugin, you get a sample rule for the administrator under the Price Rule tab. Here, you can start creating new rules for different user roles.

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Role-Based Pricing for WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type **"MWB Role Based Pricing for WooCommerce"** and click Search Plugins. Once you've found the MWB Role Based Pricing for WooCommerce plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking **"Install Now"**.

The manual installation method involves downloading our MWB Role Based Pricing for WooCommerce and uploading it to your webserver via your favorite FTP application. The WordPress codex contains [**Instructions on how to do this here**](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

= From where can I set the prices for various user roles? =
Admin can set it from **WooCommerce> MWB ROLE BASED PRICING FOR WOOCOMMERCE> User Role Setting> Check/Uncheck** the option according to the need.

= From where can I Change the text appearing next to the regular, sale, or role based price? =
Admin can change the text that appears next to the various prices.
Please follow this:
**WooCommerce< MWB ROLE BASED PRICING FOR WOOCOMMERCE> Text Setting.**

= Can we create more than one rule? =
Yes, by using this role based pricing plugin you can create more than one rule as per your need.


= How does the priority of the Rule type work in a role based pricing plugin? =
When you set more than one rule then you can set priority for the rule, with this you can set which rule you want to prioritize

= What is the use of priority in the rules? =
Admin can set the priority of the rules like an example, if the priority is 3, then the rules with 1st and 2nd priority will be applied first, and if they are not applicable, then the 3rd priority rule will be implemented for a particular user-role

= How to add my own Rule in Role Based Pricing for WooCommerce plugin? =
We have given a special tab to add your own rule as "Price Rule" where you can set Role, Priority, Discount, Rule type Role

= Can we hide the tax label for different user roles? =
Yes, Using role based pricing for WooCommerce you can hide or display the tax label for different user roles.

= Apart from Role Regular & on sale price can we set some other rules? =
Yes for this we have given an option named "Role Based Price".

= Can we have the option to apply the price rule on regular price or sale price according to our need? =
Yes, the admin can select the option to apply the price rule either on regular price or on the sale price.
